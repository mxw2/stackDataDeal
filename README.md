# 通过杜邦分析法分析上市公司近10年财报
### 使用方法:

1.首先对10年财报表格放到deal_with_data文档中，修改column_model.py文件中workbook_name字段名为excel名称，运行项目即可

2.运行完成后，在excle的"结果"sheet中会有生成的各种图(可以参考123.xlsx)

3.目前已经分析完的公司财报ppt，都放到result_ppt目录中
